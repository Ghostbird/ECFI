Detailed Description

A ring buffer structure to store register values. Instead of a read pointer and write pointer it uses indices. NOTE: If the type of the size, read, and write is changed, The arithmetic in must be checked, and casting must be reviewed to verify no wraps can occur.

regval_t* rb_read ( ringbuffer_t *  rbptr,
regval_t *  data,
uint32_t  count 
)

Read count register values from rbptr into data. The read will fail if it would overtake the write pointer. A successful read will update the read pointer. The caller is responsible for memory management of the data pointer.

Parameters
rbptrA pointer to the ring buffer struct to read from.
dataA pointer to the location where the data should be copied.
countNon-zero number of register values to read from the buffer.
Returns
The data pointer. Useful in e.g. if (rb_read(..) != NULL)
